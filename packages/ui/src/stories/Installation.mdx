# Installation

You need to install Tailwind in your project. [Tailwind Installation](https://tailwindcss.com/docs/installation)
```bash
npm install -D tailwindcss
npx tailwindcss init
```

Once `tailwind.config.js` has been created, you need to do the following:
```js
/** @type {import('tailwindcss').Config} */
module.exports = {
  content: [
    './src/your-components', // Path to your custom components
    '../../node_modules/@lifespikes/**/*.{js,ts,jsx,tsx,mdx}', // This is necessary for tailwind to compile the styles of the ui components.
  ],
  theme: {
    extend: {},
  },
  plugins: [],
}
```
Or replace the contents of your `tailwind.config` with the following [tailwind.example.config](https://github.com/lifespikes/packages/blob/main/packages/ui/tailwind.example.config.ts) file. This is required to use the theme created with the [Shadcn's theme builder](https://ui.shadcn.com/themes).


Now create a styles.css file, with the following content:
```css
@tailwind base;
@tailwind components;
@tailwind utilities;
```
